## Descripción

El objetivo de esta evaluación es consolidar los conocimientos adquiridos en las
primeras unidades del cursado (HTML, CSS, JavaScript y TypeScript), mediante la
evolución del proyecto Food Store hacia una aplicación frontend más dinámica e
interactiva.

## Consigna

A partir del proyecto desarrollado en los Trabajos Prácticos de:

- HTML
- CSS
- JavaScript
- TypeScript

Tengo que extender la aplicación actual incorporando las siguientes funcionalidades:

1. Carrito básico con persistencia
2. Busqueda y filtrado de productos

## Como ejecutar el proyecto

1. Abrir el proyecto
2. Abrir la terminal (en mi caso con `ctrl + ñ`)
3. Ejecutar el comando `pnpm i` Para instalar las dependencias del proyecto
4. Ejecutar el comando `pnpm dev` una vez instaladas las dependencias
5. Abrir el navegador y acceder a la dirección `http://localhost:5173`
